# 综述内容架构指南

## 总体结构（灵活参考，非强制框架）

综述的长度和章节分配应根据领域研究活跃度动态调整：

### 引言/背景（约15-20%）
- 选题的科学意义与工程/社会需求
- 领域核心概念界定
- 现有综述的不足与本文综述的定位
- 本综述的组织结构说明

### 方法论/技术路线（约15-25%）
- 研究方法的分类体系与演进脉络
- 不同方法流派的核心思想和适用边界
- 方法间的承继关系与变革节点

### 核心论述（约35-50%）
- 可拆为多章，每章聚焦一个子主题
- 每章应有明确的分析主线（而非文献罗列）
- 内部组织方式可选：
  - 时间演进："早期探索→方法成熟→前沿突破"
  - 方法对比："数据驱动方法→物理模型方法→混合方法"
  - 主题划分："储层表征→注入方案→长期安全性"

### 挑战与展望（约10-15%）
- 当前面临的方法论瓶颈
- 尚未解决的关键科学问题
- 跨学科融合的潜在突破方向

## 各章节写作要点

### 引言段
- 从具体科学问题切入，不要用"随着科技的发展"式套话
- 用 1-2 篇里程碑文献引出核心研究问题
- 明确阐述本综述的视角或分析框架

### 过渡段
- 章节末尾应有小结并向下一章过渡
- 过渡自然："上述方法在X场景效果显著，但在Y场景面临Z挑战。近年来，A方法因其B优势而受到关注……"

### 结论段
- 提炼核心发现，而非重复前文
- 指出当前研究的不足和未来方向
- 可以提出前瞻性判断

## 关键质量要求

1. **不单纯罗列**：每段文字背后都应该有一个分析意图——比较、递进、因果、转折
2. **引证准确**：引用编号必须与参考文献列表严格对应
3. **语言精炼**：删除冗余表述，每句话都应承载有效信息
4. **逻辑闭环**：从提出问题→分析问题→得出结论，形成完整论述链
