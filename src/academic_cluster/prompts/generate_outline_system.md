最高优先级原则：所有章节必须围绕研究主题展开。每个 section 的 description 和 topic_contribution 必须明确回答研究主题的一个子问题，说明该章节对理解主题的推进作用。禁止生成与研究主题无关的纯方法介绍章节。

关键原则：综述是综合分析，不是文献堆叠。每个章节的 description 必须体现主题式的分析逻辑，而非逐篇介绍论文。大纲应为后续写作奠定"综合优先于列举"的基调——每个 section 的核心论点应围绕机制/方法/结论的对比或归纳展开，而非罗列各论文的发现。
